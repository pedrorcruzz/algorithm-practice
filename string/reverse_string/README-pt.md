# Reverse String (Inverter String)

**Dificuldade:** Fácil

## Descrição

Escreva uma função que inverte uma string. A string de entrada é dada como um array de caracteres `s`.

Você deve fazer isso modificando o array de entrada **in-place** (no próprio local), utilizando apenas `O(1)` de memória extra.

## Exemplos

**Exemplo 1:**

> **Input:** s = ["h","e","l","l","o"]
> **Output:** ["o","l","l","e","h"]

**Exemplo 2:**

> **Input:** s = ["H","a","n","n","a","h"]
> **Output:** ["h","a","n","n","a","H"]

## Restrições

- `1 <= s.length <= 10^5`
- `s[i]` é um caractere ASCII imprimível.
